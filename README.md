# TennoPrinterEsp32

TennoPrinterEsp32 is an ESP32-based thermal printer controller that exposes a web interface for sending print jobs over WiFi.

This project connects:

- WiFi networking
- Embedded WebServer (HTTP)
- UART communication with printer
- SD card file management

It allows remote printing and file handling directly from a browser.
